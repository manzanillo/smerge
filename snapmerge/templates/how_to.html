{% trans "How to Smerge..." %}

{% trans "Smerge is a version control system for the block-based language Snap!, " %} {% trans "which enables collaboration and tinkering in the classroom. " %} {% trans "It's tailored to novice programmers and therefore easy to use for teachers and students. " %} {% trans "What we wanted to avoid was making a feature-overloaded git clone." %}

{%trans "The use of Smerge is also explained in this video (English subtitles available) in addition to the following instructions" %}

{% trans "Attention!" %}

{%trans "For Smerge to work, JavaScript extensions must be enabled in Snap! To do this, simply click once on the gear in Snap! the first time you open a Smerge project, and check the appropriate box!" %}

{% trans "Our Badger's sett" %}

{% trans "A Badger works on his sett, containing a lot of tunnels and chambers, with all his family members and is gradually expanding it over years." %}

{% trans "We proceed in exactly the same way while programming: our progress and the further development of the program from version to version we can also present as a "badger's sett". " %} {% trans "In this example, we are developing asteroids, the game in which you have to destroy all asteroids with your starship before they hit the ship." %}

{% trans "But how do we come to such a "badger's burrow"? " %} {% trans "We can open any of the versions directly in Snap! with a double click. " %} {% trans "Then we can work on our project, as we are used to." %}

{% trans "To save the changes in Snap!, we have to "post" them to our badgers sett. " %} {% trans "This is done via the corresponding block post to "smerge..." under Variables. " %} {% trans "We can summarize what we have done; this way, we will find this particular version of your project later again. " %} {% trans "Then we can find them in our badgers sett on smerge.org." %}

{% trans "Working alone is boring..." %}

{% trans "A badger does not work alone on his construction, but with the whole family. Everyone digs their own tunnel." %}

{% trans "It's the same with us when we work collaboratively. We split the work: You work on making the spaceship able to shoot, while a friend or team member handles the asteroids flying through space." %}

{% trans "As you see, everyone is working in his own "tunnels"!" %}

{% trans "At a certain point the badger wants to meet up with his family members. In the same way, we want to merge our changes to the project. Eventually we want to play the game, don't we?" %}

{% trans "Therefore we simply click the "merge"-Button in the bottom right, tick all the versions we would like to merge, and click yes to confirm." %}

{% trans "Now we will find a new version, with all changes merged. But what happens if more than one made changes to the same script? If such a situation occurs, we will have to manually choose with version we want to keep:" %}

{% trans "Go back if you made a mistake" %}

{% trans "Sometimes the badger abandons one tunnel, e.g. due to a lack of stability. He then has to go back and up in his sett, and work on from a different chamber. In the same way, sometimes we have to go back to an older version, e.g. because we made some mistakes or tried something that didn't really work out. In order to do so, we simply open the version we want to rever to, open it in Snap! by double-clicking, and work on from there. This way, we will get a new "tunnel"." %}

{% trans "Kanban board" %}

{% trans "Group work requires communication between participants, for this propose Smerge provides a Kanban board. It can be opened with a button on the top right hand corner below the setting button. Cards can be added by pressing the ADD button at the top of a column, and to edit them click on the pencil icon of a card, write some text and click on the save icon when done. Additionally, the color of a card can be changed by clicking on the palette icon of a card. They can be moved to a different column by dragging." %}

{% trans "To change the column layout, first press the button at the top left hand corner and then add, remove, rename or drag columns. You can have up to five columns in total." %}

{% trans "Built for teachers and the classroom" %}

{% trans "What does Smerge offer you and how it can help with teaching?" %}

  • {% trans "enabling collaboration for project-based learning with Snap!" %}
  • {% trans "all the data is always available and automatically backed up, so no more "I forgot my stuff at home" or "my partner is ill, I can't work" from your students." %}
  • {% trans "set up templates in the sense of "code skeletons", where you can supply your students with custom blocks, some code pieces already in place, ... You can even add further templates via drag and drop." %}
  • {% trans "the development process is always visible, to your students as well as to you. This allows for assessing individual contribution and giving individual feedback." %}
  • {% trans "you see the version history and are able to revert changes and go back to old versions, which enables tinkering and playing around in a risk-free enviroment." %}

{% trans "Login and User Management" %}

{% trans "We deliberatly have no dedicated user accounts or management for the average user. Instead, to secure your projects, you will get a PIN on the time of project creation, and can set a password. If you add your e-mail at project creation, you can retrieve your PIN and password for all your projects if you forgot them." %}
